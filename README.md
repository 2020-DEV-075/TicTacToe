# TicTacToe
> iOS Swift TicTacToe Application

## Requirements

- iOS 10.0+
- Xcode 11 or later
- Swift 5.1

## Dependencies

No dependecies used in this project. 

## How to run the run the app?

Clone the `main` or `develop` branch, open the project in the correct Xcode version and run.
